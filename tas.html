Trainer Sensor berbasis arduino UNO & NodeMCU sebagai media pembelajaran Teknik Kontrol Sistem Mekatronik di Kompetensi Keahlian Teknik Mekatronika SMK N 1 Bawang

Trainer Sensor

Apa itu Trainer Sensor berbasis Arduino UNO dan NodeMCU?

Tariner ini adalah sebuah media pembelajaran berupa Trainer yang dibuat dari susunan dari beberapa sensor, dan output serta mikrokontroler yang bertujuan untuk memudahkan proses pembelajaran pada mata pelajaran teknik kontrol sistem mekatronik.

Apa saja komponen utamanya?

Komponen sensor :

1. Sensor Ultrasonic
2.Sensor LDR
3.Sensor Gas MQ-2
4. Sensor Suhu LM35
5. Sensor Api
6. Sensor DHT11

Komponen Output:

1. LED
2.Module Relay
3.LCD 16X2
4. Buzzer
5. Motor Servo
6. Lampu

Mikrokontroler:

1. Arduino UNO R3
2.NodeMCU Esp8266

Apa saja kelebihannya?

1. Mendukung pembelajaran pemrograman Mikrokontroler dasar dengan menggunakan Sensor disertai output/aktuator.
2. Mendukung pengembangan kemampuan tingkat lanjut dengan penerapan Internet of Things
3. Materi dan Labsheet dibuat dengan mempertimbangkan Kompetensi Dasar Teknik Kontrol Sistem Mekatronik
4. Desain tata letak dibuat dengan memperhatikan kenyamanan untuk meningkatkan motivasi belajar.

Buku Pendamping Trainer

Buku Pendamping ini berisi uraian materi yang berkaitan dengan praktikum, petunjuk dan cara menggunakan Trainer, Software yang berkaitan dengan praktikum serta yang terakhir adalah Job Sheet yang digunakan untuk praktikum. dengan adanya Buku Pendamping ini diharapkan proses praktikum berjalan sesuai prosedur serta memberikan dampak yang positif dalam pembelajaran.

Berikut adalah Labsheet yang terdapat pada Buku Pendamping :
1. Kontrol LED menggunakan mikrokontroler Arduino UNO
2. Monitoring suhu ruang dengan sensor LM35
3. Kontrol lampu otomatis dengan sensor LDR
4. Alarm kebakaran menggunakan sensor flame
5. Kontrol servo dengan sensor Ultrasonic HCSR-04
6. Deteksi gas dengan sensor MQ-2
7. Kendali lampu berbasis Internet of Things
8. Kendali servo berbasis Internet of Things
9. Monitoring suhu dan kelembaban berbasis Internet of Things

Untuk dapat memberikan penilaian pada Media Pembelajaran Trainer Sensor ini, sebaiknya anda membaca Buku Pendamping Trainer terlebih dahulu.
