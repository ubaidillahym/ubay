Trainer Sensor berbasis arduino UNO & NodeMCU sebagai media pembelajaran Teknik Kontrol Sistem Mekatronik di Kompetensi Keahlian Teknik Mekatronika SMK N 1 Bawang

Trainer Sensor

Apa itu Trainer Sensor berbasis Arduino UNO dan NodeMCU?

Yaitu sebuah media pembelajaran berupa Trainer yang dibuat dari susunan dari beberapa sensor, dan output serta mikrokontroler yang bertujuan untuk memudahkan proses ppembelajaran pada mata pelajaran teknik kontrol sistem mekatronik.

Apa saja komponen utamanya?

Komponen sensor :

1. Sensor Ultrasonic
2.Sensor LDR
3.Sensor Gas MQ-2
4. Sensor Suhu LM35
5. Sensor Api 6. Sensor DHT11

Komponen Output:

1. LED
2.Module Relay
3.LCD 16X2
4. Buzzer
5. Motor Servo
6. Lampu

Mikrokontroler:

1. Arduino UNO R3
2.NodeMCU Esp8266

Apa saja kelebihannya?

1. Mendukung pembelajaran pemrograman Mikrokontroler dasar dengan menggunakan Sensor dan beserta output
2. Mendukung pengembangan kemampuan tingkat lanjut dengan penerapan Internet of Things
3. Materi dan Labsheet dibuat dengan mempertimbangkan Kompetensi Dasar Teknik Kontrol Sistem Mekatronik
4. Desain tata letak dibuat dengan memperhatikan kenyamanan serta motivasi belajar
